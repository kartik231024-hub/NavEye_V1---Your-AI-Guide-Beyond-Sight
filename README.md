# NavEye_V1---Your-AI-Guide-Beyond-Sight
AI-powered wearable guide for the blind and visually impaired — real-time object detection, OCR text reading, face recognition, conversational AI, and one-press SOS alerts, running on ESP32-S3 + ESP32-CAM with a Gemini-powered backend. Solo-built: firmware, circuit, and 3D-printed enclosure.
